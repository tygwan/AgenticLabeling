# Dev Standards Adapter Drafts

This directory contains the first-pass adapter set for `dev-standards`.

Adapters translate the standards into concrete tools or ecosystems.

They do not redefine the core.

## Intent

Adapters should:

- operationalize the core
- remain tool-specific
- stay replaceable

If an adapter contains guidance that would remain true without the tool, that guidance likely belongs in the core or a profile instead.
