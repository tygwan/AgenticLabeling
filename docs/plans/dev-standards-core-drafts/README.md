## Draft Intent

These are first-pass, one-page rules.

Each draft is meant to answer:

- what the rule requires
- why the rule exists
- how to apply it
- what it does not prescribe

The next refinement step is to normalize language, rule strength, and enforcement style across all twelve drafts.
