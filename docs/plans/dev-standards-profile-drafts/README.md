# Dev Standards Profile Drafts

This directory contains first-pass drafts for optional `dev-standards` practice profiles.

Profiles are not universal requirements.

They are activated only when a project shape justifies stricter expectations than the core alone.

## Intent

These drafts exist to:

- keep the core small
- preserve stricter guidance where it genuinely matters
- prevent context-heavy rules from polluting the universal standard
